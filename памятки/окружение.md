## Деплой

**Источник истины — GitHub.** Деплой ручной в два шага: push в master, потом `git pull` на сервере.

### Шаг 1: запушить в GitHub
```bash
git push origin master
```
> Если SSH к `github.com:22` режется (бывает на этом провайдере/VPN), пушить через HTTPS:
> ```bash
> git push https://github.com/anvargsm22-oss/altay-eco-trade.git master
> ```

### Шаг 2: подтянуть на сервере
```bash
ssh anvar-main 'cd /opt/apps/site && git pull --ff-only origin master'
```

Сайт обновляется **мгновенно** — статика, nginx ничего не кеширует, рестарт не нужен.

> ⚠️ Никакого webhook / cron / git-hook'а для авто-выкатки **нет**. Пока этот шаг не сделан — на altayai.ru висит старая версия. Проверено 19.06.2026: коммит на GitHub без ручного pull может висеть невидимым для сайта сколь угодно долго.

### Проверка после деплоя
```bash
curl -s "https://altayai.ru/contacts.html?v=$(date +%s)" | grep -E "tel:|mailto:"
```
или через WebFetch с антикеш-параметром.
